## Getting Started

**To run the app:**

```zsh
docker-compose up --build
```

**To run the server-side tests:**

```zsh
docker-compose -f docker-compose.test.yml up --build
```

**To run the client-side tests:**

```zsh
cd client
npm install
npm test
```
